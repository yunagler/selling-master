### ElevenLabs Pro ($22/חודש) - קול עברי
- Multilingual v2 - העברית הכי טבעית

### עלות כוללת: ~$500 ל-62 שיעורים (22 שעות)

## תהליך הפקה לכל שיעור

```
1. תסריט (MD) ← כתוב ומוכן
2. ElevenLabs API → אודיו MP3 (עברית, קול טבעי)
3. שקפים HTML → PNG screenshots (1920x1080)
4. HeyGen Studio:
   - אווטר מותאם (מהתמונה שלנו)
   - שקפים כרקע / Picture-in-Picture
   - אודיו מ-ElevenLabs כקלט
   - ייצוא 4K
5. עריכה סופית (Descript/CapCut):
   - כתוביות
   - מוזיקת רקע
   - אפקטי מעבר
```

## בדיקת איכות לפני ייצור מלא

**חובה לעשות פיילוט:**
1. הרשם ל-HeyGen Free (3 סרטונים)
2. צור אווטר מהתמונה שלנו
3. ייצר שיעור 1.1 (10 דקות) כמבחן
4. בדוק: lip sync בעברית, טבעיות, איכות תמונה
5. אם מספק → הרשם ל-Pro ותתחיל ייצור
