### מה זה SEO — בפשטות (SLIDE: Google → חנות שלך → לקוח)

SEO — Search Engine Optimization. אופטימיזציה למנועי חיפוש.

במילים פשוטות: לגרום לגוגל להציג את החנות שלכם כשמישהו מחפש מוצר שאתם מוכרים.

**דוגמה:** מישהו כותב בגוגל "כרית ארגונומית לצוואר". אם עשיתם SEO נכון — החנות שלכם מופיעה בתוצאות. הוא לוחץ. מגיע לחנות. קונה. בלי שהוצאתם שקל על פרסום.

**שלושה רכיבים של SEO:**
1. **On-Page** — מה שנמצא באתר שלכם (כותרות, תוכן, תמונות)
2. **Off-Page** — מה שנמצא מחוץ לאתר (קישורים, אזכורים)
3. **טכני** — מהירות, מובייל, מבנה אתר

בשיעור הזה — On-Page וטכני. כי זה מה שאתם שולטים בו מהיום הראשון.

### שלב 1: מחקר מילות מפתח (SLIDE: כלים למחקר)

לפני שמתחילים — צריך לדעת מה אנשים מחפשים.

**כלים חינמיים:**

**Google Keyword Planner** — חינם עם חשבון Google Ads. מראה כמה אנשים מחפשים כל מילה בחודש. ומציע מילות מפתח קשורות.

**Google Autocomplete** — התחילו לכתוב בגוגל ותראו מה הוא משלים. "כרית ארגונומית..." → "כרית ארגונומית לצוואר", "כרית ארגונומית IKEA", "כרית ארגונומית מומלצת." כל השלמה = מילת מפתח שאנשים מחפשים.

**Ubersuggest** — כלי חינמי (עד 3 חיפושים ביום). נפח חיפוש, רמת תחרות, ורעיונות.

**Google Trends** — מגמות חיפוש. עולה? יורד? עונתי?

**הכלל:** חפשו מילות מפתח עם:
- נפח חיפוש סביר (100-5,000 חיפושים בחודש בישראל)
- תחרות נמוכה-בינונית
- כוונת רכישה (לא "מה זה כרית ארגונומית" — אלא "לקנות כרית ארגונומית")

**דוגמה:** "שמלת ערב" — 12,000 חיפושים בחודש. תחרות גבוהה מאוד. "שמלת ערב ירוקה מידה L" — 200 חיפושים. תחרות נמוכה. סיכוי הרבה יותר גבוה להגיע לעמוד ראשון.

**מילות מפתח Long Tail** — ארוכות וספציפיות. פחות חיפושים, אבל הרבה יותר ממירות. "ארגונית למגירת מטבח IKEA" > "ארגוניות" — כי מי שמחפש את הראשון — מוכן לקנות.

### שלב 2: אופטימיזציית כותרות (SLIDE: Title Tag + Meta Description)

**Title Tag — כותרת העמוד בגוגל.**

זה הדבר הכי חשוב ב-SEO. גוגל מסתכל על הכותרת כדי להבין על מה העמוד.

**נוסחה:** [מילת מפתח ראשית] — [תכונה/יתרון] | [שם החנות]

**דוגמאות:**
- "כרית ארגונומית לצוואר — Memory Foam | SleepWell"
- "סט סכינים יפניות — פלדת דמשק, 5 סכינים | שף בבית"
- "מארגן מגירות מבמבוק — 8 תאים | OrderHome"

**אורך:** 50-60 תווים. יותר מזה — גוגל חותך.

**Meta Description — תיאור בגוגל.**

זה הטקסט שמופיע מתחת לכותרת בתוצאות החיפוש. לא משפיע ישירות על דירוג, אבל משפיע על **שיעור הלחיצה**.

**נוסחה:** [מה המוצר] + [למה לקנות אצלכם] + [CTA]

"כרית ארגונומית Memory Foam — תמיכה מושלמת לצוואר. משלוח חינם. אחריות 3 שנים. הזמינו עכשיו!"

**אורך:** 120-155 תווים.

**ב-Shopify:** כל עמוד מוצר → "Search engine listing preview" → ערכו את ה-Title וה-Description.

### שלב 3: URL נקי (SLIDE: URL טוב vs גרוע)

**URL גרוע:** yourstore.co.il/products/p12345-item-new-2026
**URL טוב:** yourstore.co.il/products/karit-ergonomit-memory-foam

**הכללים:**
- מילת מפתח ב-URL
- באנגלית (עברית ב-URL הופכת ל-encode מוזר)
- מקפים בין מילים (לא קו תחתון)
- קצר — 3-5 מילים
- בלי מספרים או קודים

**ב-Shopify:** בעת יצירת מוצר — הגדרות SEO → URL handle. שנו ידנית.

### שלב 4: אופטימיזציית תמונות (SLIDE: ALT text + גודל קובץ)

גוגל לא "רואה" תמונות. הוא קורא את ה-ALT text.

**ALT text — תיאור חלופי:**
- תארו את התמונה + מילת מפתח
- "כרית ארגונומית Memory Foam לצוואר — צבע לבן" — טוב
- "IMG_3847.jpg" — גרוע
- "כרית כרית כרית ארגונומית כרית שינה" — ספאם. גרוע.

**שם קובץ:**
- שנו את שם הקובץ לפני העלאה
- "karit-ergonomit-memory-foam.jpg" — טוב
- "IMG_3847.jpg" — גרוע

**גודל קובץ:**
- תמונות כבדות = אתר איטי = דירוג נמוך
- דחסו ל-100-200KB לתמונה
- כלים: TinyPNG, Squoosh, ShortPixel
- פורמט WebP — 25-35% קטן יותר מ-JPEG. Shopify תומך.

**דוגמה ישראלית:** "HomeLab" — חנות מוצרי בית. שיפרו ALT text ל-200 תמונות. תוך 3 חודשים — 40% עלייה בתנועה מ-Google Images. בחינם.

### שלב 5: Schema Markup (SLIDE: Rich Snippets בגוגל)

Schema זה קוד שמספר לגוגל מה בדיוק נמצא בעמוד. כשגוגל מבין — הוא מציג "Rich Snippets" — כוכבים, מחיר, זמינות.

**למה זה חשוב?** תוצאה עם כוכבים ומחיר בולטת יותר. שיעור לחיצה — גבוה ב-30%.

**Product Schema כולל:**
- שם מוצר
- מחיר
- זמינות (במלאי / אזל)
- דירוג (כוכבים)
- תמונה

**ב-Shopify:** רוב התבניות כוללות Schema אוטומטית. אבל תבדקו עם הכלי של גוגל — Google Rich Results Test. מזינים URL ובודקים.

**ב-WooCommerce:** תוסף Yoast SEO מוסיף Schema אוטומטית. בדקו שזה מופעל.

### שלב 6: מהירות אתר (SLIDE: כל שנייה = 7% פחות המרות)

שנייה אחת נוספת בזמן טעינה = 7% ירידה בהמרות.

**איך בודקים:** Google PageSpeed Insights. מזינים את הכתובת. מקבלים ציון 0-100. מעל 90 = מעולה. 50-89 = צריך שיפור. מתחת ל-50 = בעיה.

**מה מאט את האתר:**
- תמונות כבדות — דחסו!
- יותר מדי אפליקציות — הסירו מה שלא משתמשים
- פונטים כבדים — השתמשו ב-Google Fonts
- קוד מיותר — theme קל = אתר מהיר

**פעולות מיידיות:**
1. דחסו כל תמונה ל-200KB מקסימום
2. הסירו אפליקציות שלא משתמשים
3. הפעילו lazy loading — תמונות נטענות רק כשגוללים אליהן
4. השתמשו ב-CDN — Shopify כולל Cloudflare CDN מובנה

### שלב 7: תוכן — בלוג (SLIDE: בלוג = תנועה אורגנית)

בלוג הוא מכונת SEO.

**כל פוסט = עמוד חדש שגוגל יכול לדרג.** 10 פוסטים = 10 הזדמנויות להופיע בחיפוש.

**מה לכתוב:**
- מדריכים — "איך לבחור כרית ארגונומית — 5 טיפים"
- השוואות — "כרית Memory Foam vs כרית לטקס — מה עדיף?"
- רשימות — "10 מוצרי המטבח שכל בית צריך ב-2026"

**כל פוסט כולל:**
- מילת מפתח בכותרת
- 800-1,500 מילים
- תמונות עם ALT
- קישור למוצרים בחנות

**דוגמה ישראלית:** "iHerb ישראל" — לא באמת iHerb, אבל בלוגים ישראליים שכותבים על מוצרי iHerb מושכים אלפי מבקרים מגוגל. אותו עיקרון — תכתבו תוכן רלוונטי, אנשים ימצאו אתכם.

### סיכום (SLIDE: "7 שלבים ל-SEO שעובד")

1. **מחקר מילות מפתח** — דעו מה אנשים מחפשים
2. **כותרות** — Title Tag עם מילת מפתח
3. **URL** — נקי, באנגלית, עם מילת מפתח
4. **תמונות** — ALT text, שמות קבצים, דחיסה
5. **Schema** — Rich Snippets בגוגל
6. **מהירות** — מהיר = דירוג גבוה
7. **בלוג** — תוכן = תנועה

SEO זה לא ספרינט. זה מרתון. התוצאות לא מגיעות מחר. הן מגיעות תוך 2-4 חודשים. אבל כשהן מגיעות — הן מגיעות בחינם. כל יום. כל חודש. לנצח.

תתחילו היום.
