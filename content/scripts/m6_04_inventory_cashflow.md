### פתיחה (SLIDE: "82% מהעסקים שנסגרים — נסגרים בגלל תזרים מזומנים")

שמונים ושניים אחוז.

לא בגלל שהם לא מוכרים. לא בגלל שאין ביקוש. בגלל שנגמר להם הכסף. מכירות של 100,000 בחודש? מעולה. אבל אם שילמתם לספקים 60,000, על פרסום 15,000, על מלאי 30,000 — ואין לכם 5,000 לשכירות? סגור.

תזרים מזומנים הוא **ההבדל בין כסף שנכנס לכסף שיוצא — ומתי**. ותזמון הוא הכל.

אני LEO. השיעור הזה לא סקסי. הוא לא מרגש. הוא הכי חשוב בקורס.

### למה מלאי הורג עסקים (SLIDE: ציר זמן — קניית מלאי מול קבלת תשלום)

הבעיה הקלאסית:

**יום 1:** מזמינים מלאי מסין. 20,000 שקל. משלמים.
**יום 30:** המלאי מגיע.
**יום 31-60:** מתחילים למכור.
**יום 60-90:** כסף מתחיל להיכנס.

**30-90 יום** שהכסף שלכם תקוע במלאי. ובינתיים — שכירות. פרסום. מע"מ. ביטוח לאומי. חיים.

**דוגמה ישראלית:** בחור ייבא מסין מוצרי ספורט. הזמין ב-80,000 שקל. משלוח 2 חודשים. הגיע. התחיל למכור. אבל בינתיים? הפרסום עלה 10,000. השכירות 5,000. אין כסף לפרסם. המלאי יושב. ובלי פרסום — אין מכירות. ובלי מכירות — אין כסף. מעגל קסמים שלילי.

### כמה מלאי להזמין (SLIDE: נוסחת מלאי אופטימלי)

יש נוסחה. פשוטה.

**מלאי אופטימלי = (מכירות יומיות ממוצעות × זמן אספקה) + מלאי ביטחון**

**דוגמה:**
- מוכרים 5 יחידות ביום
- זמן אספקה מהספק: 14 יום
- מלאי ביטחון: 20% (למקרה שהספק מאחר)

**חישוב:** (5 × 14) + 20% = 70 + 14 = **84 יחידות**

לא 500. לא 1,000. 84. ואז מזמינים שוב.

**כלל:** מזמינים מלאי ל-30-45 יום. לא ל-6 חודשים. כסף במלאי = כסף לא זמין.

### 3 טכניקות לניהול מלאי (SLIDE: 3 כרטיסים)

**טכניקה 1: מכירה מוקדמת (Pre-order)**

מוכרים **לפני** שהמלאי מגיע. "מוצר חדש! זמין בעוד שבועיים. הזמינו עכשיו ב-15% הנחה."

**יתרון:** הכסף נכנס לפני שאתם משלמים לספק. תזרים חיובי.

**דוגמה ישראלית:** מותג ביגוד ישראלי שמשיק קולקציה חדשה. שבועיים לפני — פותחים pre-order. 200 הזמנות. 60,000 שקל נכנסים. מזמינים מלאי ב-25,000. רווח + תזרים חיובי.

**טכניקה 2: דרופשיפינג כגיבוי**

המוצרים העיקריים — מלאי. מוצרים חדשים שבודקים? דרופשיפינג. אפס סיכון. אם נמכר — מזמינים מלאי. אם לא — מורידים.

**טכניקה 3: תנאי תשלום מהספק**

שוטף + 30. שוטף + 60. אשראי.

**מה זה אומר?** קונים מלאי היום. משלמים בעוד 30-60 יום. בינתיים — מוכרים. הכסף מהמכירות נכנס לפני שמשלמים לספק. תזרים מושלם.

**איך מקבלים תנאים כאלה?** בהתחלה — קשה. ספק חדש רוצה תשלום מיידי. אבל אחרי 3-4 הזמנות מוצלחות — בקשו. "הזמנתי 4 פעמים ושילמתי בזמן. אפשר שוטף + 30?" רוב הספקים יסכימו.

**דוגמה ישראלית:** ספק מוצרי ניקיון נותן ללקוחות ותיקים שוטף + 30. הלקוח מזמין ב-1 לחודש. מוכר עד ה-15. מקבל כסף. משלם לספק ב-30. אף פעם לא "מינוס."

|------|--------|-----------|-------|-------------|-----------|-----------|------|
| 1 | | | | | | | |
| 2 | | | | | | | |
| 3 | | | | | | | |

**הכנסות:** כמה כסף באמת נכנס (לא מכירות — כסף שהתקבל).
**עלות מלאי:** כמה שילמתם על מוצרים.
**פרסום:** כל הוצאות השיווק.
**הוצאות קבועות:** שופיפיי, כלים, שכירות, ביטוח.

**הכלל:** אם היתרה יורדת 3 חודשים ברציפות — יש בעיה. עצרו. בדקו. תקנו.

### חוק ה-30/30/30 (SLIDE: עוגה מחולקת ל-3)

חלוקת הכנסות בריאה:

**30% — עלות מוצר.** כולל מלאי, משלוח מספק, מכס, מע"מ יבוא.
**30% — שיווק ומכירות.** פרסום, כלים, שותפים.
**30% — הוצאות תפעול + רווח.** שופיפיי, עובדים, כלים, ומה שנשאר = רווח.
**10% — חירום.** תמיד. אף פעם לא להשתמש. אלא אם באמת חירום.

**אם עלות המוצר אוכלת 50%?** המחיר נמוך מדי. או הספק יקר מדי. תקנו.
**אם השיווק אוכל 50%?** הקמפיינים לא יעילים. אופטימיזציה.
**אם אין 10% חירום?** אתם הולכים על חבל דק. ספק שמאחר. גוגל שחוסם. מע"מ שמפתיע. בלי כרית — קורסים.

### מע"מ — המלכודת (SLIDE: "מע"מ = לא שלכם")

שמעו טוב.

**17% מהמכירות שלכם — לא שלכם.** זה מע"מ. של המדינה. אתם רק שומרים על זה. וכל חודשיים — צריך להעביר.

**הטעות הנפוצה:** "מכרתי ב-100,000 שקל! יש לי 100,000!" — לא. יש לכם 85,470 (לפני מע"מ). ו-14,530 שייך למע"מ.

**הפתרון:** חשבון בנק נפרד. כל מכירה — 17% עוברים מיד לחשבון המע"מ. אל תיגעו בזה. כשמגיע הדיווח — הכסף שם.

**דוגמה ישראלית:** בעל חנות אונליין מכר ב-300,000 שקל ברבעון. שילם על מלאי, פרסום, כלים. כשהגיע דיווח מע"מ — 42,000 שקל. "מאיפה אני מביא?" הלוואה. ריביות. לחץ. כל זה כי לא שם מע"מ בצד.

### כלים שעוזרים (SLIDE: 4 כלים)

**Excel/Google Sheets** — טבלת תזרים חודשית. חינמי. פשוט. 5 דקות בחודש.

**Inventory Planner** (Shopify App) — מחשב אוטומטית כמה מלאי להזמין. מבוסס על היסטוריית מכירות. 79 דולר/חודש.

**Stocky** (Shopify) — מובנה ב-Shopify Plus. ניהול מלאי ותחזיות.

**חשבונית ירוקה / iCount / Rivhit** — הנהלת חשבונות. מע"מ אוטומטי. חשבוניות. דוחות.

### סיכום (SLIDE: "כסף במלאי = כסף קפוא. כסף בבנק = חירות.")

3 כללים שישמרו עליכם:

**1. מלאי ל-30-45 יום.** לא יותר.
**2. 30/30/30/10.** עלות, שיווק, תפעול+רווח, חירום.
**3. מע"מ בצד. תמיד.**

תזרים חיובי = עסק בריא. תזרים שלילי = עסק בסכנה. זה לא משנה כמה אתם מוכרים.

שיעור הבא — שירות לקוחות מעולה. איך שירות טוב הופך ללקוחות חוזרים. ולמה לקוח מרוצה שווה 10 לקוחות חדשים.
